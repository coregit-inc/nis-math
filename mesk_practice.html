Two's Complement (8-bit)
Practice
Score: 0
Express 0 as 8-bit two's complement binary.
Method for negative numbers: (1) write |n| in binary, (2) flip all bits (one's complement), (3) add 1. Range for 8-bit two's complement: −128 to +127.

Fixed-Point Binary
Practice
Bits 00000000 · the right 3 bits are fractional. What's the unsigned decimal value?
Method: integer part = left bits as usual. Fractional part: each bit right of the point has weight 2⁻¹=0.5, 2⁻²=0.25, 2⁻³=0.125, 2⁻⁴=0.0625…
Floating-Point: Mantissa & Exponent
Practice
Represent 0 as a signed two's-complement floating-point (12-bit mantissa, 4-bit exponent).


Method:
  1. Convert |n| to binary (incl. fractional part)
  2. Normalise: shift so the binary point comes after the leading 0.1 (positive) or 1.0 (negative two's complement)
  3. Mantissa = the bits, exponent = number of shifts
  4. Negative numbers: take two's complement of the mantissa

Boolean Laws

LawForm
IdentityA · 1 = A · A + 0 = A
NullA · 0 = 0 · A + 1 = 1
IdempotentA · A = A · A + A = A
ComplementA · ¬A = 0 · A + ¬A = 1
Involution¬¬A = A
CommutativeA · B = B · A
AssociativeA · (B · C) = (A · B) · C
DistributiveA · (B + C) = A·B + A·C
De Morgan 1¬(A · B) = ¬A + ¬B
De Morgan 2¬(A + B) = ¬A · ¬B
AbsorptionA + A·B = A · A·(A+B) = A

Worked: ¬((A + ¬B) · (C + ¬D))

¬((A + ¬B) · (C + ¬D))
= ¬(A + ¬B) + ¬(C + ¬D)     // De Morgan 1
= (¬A · ¬¬B) + (¬C · ¬¬D)   // De Morgan 2
= ¬A·B + ¬C·D                // Involution

Worked: A·B + A·¬B

A·B + A·¬B
= A·(B + ¬B)     // Distributive (factor out)
= A·1            // Complement
= A              // Identity

Worked: A + A·B

A + A·B
= A·(1 + B)      // Distributive
= A·1            // Null (1+anything = 1)
= A              // Identity
// (Or apply Absorption directly)

SELECT S.Title, SUM(D.Product_count) AS total

function isEven(n) {

2. Sum all even numbers from 1 to 100. 
let sum = 0;
for (let i = 2; i <= 100; i += 2) {
    sum += i;
}

3. Find the maximum value in [3,7,2,9,4,1,8]. 
const arr = [3,7,2,9,4,1,8];
let max = arr[0];
for (let i = 1; i < arr.length; i++) {
    if (arr[i] > max) max = arr[i];
}
console.log(max); // 9
4. Reverse a string without using .reverse(). 
function reverse(s) {
    let r = "";
    for (let i = s.length - 1; i >= 0; i--) {
        r += s[i];
    }
    return r;
}
5. Count vowels in a string. 
function countVowels(s) {
    const vowels = "aeiouAEIOU";
    let count = 0;
    for (let ch of s) {
        if (vowels.includes(ch)) count++;
    }
    return count;
}

PHP Patterns

// server-side reference
Note: PHP runs on the server — it can't execute in your browser. Below are the exact patterns you'll need to write on paper.
Database Connection (2025 Q8)
Pattern
<?php
$servername = "localhost";
$username   = "username";
$password   = "password";
$db         = "library";

$conn = mysqli_connect($servername, $username, $password, $db);

if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}
echo "Connected successfully";
?>
Receiving Form Data
Pattern
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $name    = $_POST["name"];
    $message = $_POST["message"];

    echo "Hello, " . htmlspecialchars($name);
}
?>
Always use htmlspecialchars() when echoing user input — prevents XSS attacks.
Running an SQL Query
Pattern
<?php
$sql = "SELECT id, name FROM users WHERE active = 1";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
    while ($row = mysqli_fetch_assoc($result)) {
        echo $row["id"] . " - " . $row["name"] . "<br>";
    }
} else {
    echo "No results";
}

mysqli_close($conn);
?>
Insert with Prepared Statement (safe)
Pattern
<?php
$stmt = mysqli_prepare($conn, "INSERT INTO users (name, age) VALUES (?, ?)");
mysqli_stmt_bind_param($stmt, "si", $name, $age);  // s=string, i=int
mysqli_stmt_execute($stmt);
mysqli_stmt_close($stmt);
?>
