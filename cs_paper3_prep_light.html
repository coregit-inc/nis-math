Бинарный поиск + сложность
Заполнение пропусков в коде, плюс вопросы про временную сложность O(log N) и максимальное число итераций для массива размера N.
Ключевые моменты
  • Условие цикла: low <= high (или с флагом)
  • mid = INT((low+high)/2)

Syntax / Runtime / Logic

Assembly: как заполнять trace table

Этот вопрос даёт 5-9 баллов каждый год. Алгоритм трассировки одинаковый — нужно лишь понимать каждую инструкцию. 

// Пример из экзамена 2025
00  LDA 08      // Загрузить значение из ячейки 08 в Accumulator
01  SUB 09      // Вычесть значение ячейки 09 из ACC
02  BRP 06      // Если ACC >= 0, прыгнуть на адрес 06
03  LDA 08      // Иначе: загрузить 08
04  OUT         // Вывести ACC
05  HLT         // Остановиться
06  STA 08      // Сохранить ACC в 08
07  BRA 00      // Прыгнуть на 00 (цикл)
08  DAT 6       // Данные
09  DAT 4       // Данные

Эта программа находит остаток от деления 6 на 4. Результат: 2.

→ 04 / Шаблон кода

Bubble Sort — заучить наизусть

Этот код должен выходить автоматически. Различие только в направлении сортировки (descending — знак < меняется на >). 

// Сортировка по убыванию (2025 экзамен)
FOR i ← 0 TO length(arr) - 2 DO
    FOR j ← 0 TO length(arr) - i - 2 DO
        IF arr[j] < arr[j+1] THEN     // для возрастания: arr[j] > arr[j+1]
            temp ← arr[j]
            arr[j] ← arr[j+1]
            arr[j+1] ← temp
        ENDIF
    ENDFOR
ENDFOR
Памятка
Запомнить нужно 4 элемента:
1) Два вложенных цикла
2) Сравнение соседних: arr[j] и arr[j+1]
3) Обмен через временную переменную
4) Каждый "проход" — самое большое (или маленькое) "всплывает" к концу
→ 05 / Структуры

Binary Search Tree — как строить

Дают массив. Первый элемент становится корнем. Каждый следующий: меньше — налево, больше — направо. 

// Массив из 2025 экзамена
["M", "B", "A", "K", "H", "V", "T"]

// Построение пошагово:
1. M → корень
2. B < M → B налево от M
3. A < M, A < B → A налево от B
4. K < M, K > B → K направо от B
5. H < M, H > B, H < K → H налево от K
6. V > M → V направо от M
7. T > M, T < V → T налево от V

// Формат узла: [Left ptr | Data | Right ptr]
// Указатель = индекс узла, или NULL (-1) если нет
⚠ Важно Не забывайте указатели (либо номер связанного узла, либо NULL/-1 для пустых ветвей). Без указателей баллы не даются.
→ 06 / Декларативка

Prolog — правила и запросы

% Факты (что-то является правдой)
parent_child(maryam, lisa).
parent_child(jiger, maryam).
spouse(jiger, tamila).

% Правила (если ... то ...)
parent(X, Y) :- parent_child(X, Y).

% Sibling: X и Y — родственники, если у них общий родитель
sibling(X, Y) :- 
    parent_child(Z, X),   % Z родитель X
    parent_child(Z, Y),   % AND Z родитель Y
    X \= Y.               % AND X ≠ Y

% Family: X в семье с Y, если они супруги ИЛИ родитель-ребёнок
family(X, Y) :-
    spouse(X, Y);                % OR
    parent_child(X, Y);          % OR
    parent_child(Y, X).

Запросы (queries)

flies(Y, sunqar).Кто летает рейсом sunqar?
child(jiger, Y).Дети jiger? Возвращает Y
direct(shymkent, Z).Прямые рейсы из Шымкента

На что обратить внимание

Типичные ошибки, на которых теряют баллы.

⚠ Stack operations Когда пишете список операций со стеком, значения должны быть в кавычках: push("C#"), а не push(C#). Иначе баллы не дают.
⚠ Prolog Константы пишутся с маленькой буквы (bolat, а не Bolat). С большой буквы — только переменные.
⚠ BST Не забывайте указатели (Left ptr, Right ptr). Узел без указателей — это не узел дерева, баллов не получите.
⚠ HTML radio buttons У всех radio в одной группе должен быть одинаковый name="". Иначе можно будет выбрать все одновременно — задание считается невыполненным.
⚠ Brand names Из инструкции: "No marks will be awarded for using brand names". Не пишите "Python", "Excel", "Word" в ответах — пишите "high-level language", "spreadsheet software" и т.п.
⚠ Trace table — Condition Заполняйте колонку Condition только когда выполняется BRP/BRZ. На обычных инструкциях оставляйте пустым.
⚠ Bubble sort range Внутренний цикл должен идти до n - i - 1, иначе будет выход за границы массива (runtime error).
⚠ Encrypt 'z' В алгоритме Цезаря если буква 'z', то alphabet[j+1] = alphabet[26] — runtime error (out of range). Решение: проверка if char='z' → wrap to 'a'.
